First Project - MBTI Community Web Site

MBTI COMMUNITY

MBTI, which divides a person's personality into a total of 16 types, is currently a huge issue in Korea, but on the other hand, there is no separate hub web page yet. I tried to create a web page by self-taught PHP language in the SQL language I learned at the academy. Popular bulletin boards that collect only popular articles, which are essential elements of the community, and free bulletin boards that users can freely discuss are classified according to their tendencies, but users can freely discuss all tendencies on the integrated free bulletin board if they want. I used XAMPP (Apache + MariaDB + PHP) for this project.
